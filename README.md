# 🚀 Blog App (Full Stack)

🔗 **Live Demo (Frontend):**
https://blog-appfrontend.netlify.app/

---

## 📌 Overview

This is a full-stack blog application where users can create, manage, and upload blog content with images. The project demonstrates real-world deployment, API integration, and cloud-based media handling.

---

## ⚙️ Tech Stack

### Frontend

* React (Vite)
* Fetch API
* Deployed on Netlify

### Backend

* Laravel (PHP)
* REST APIs
* Dockerized deployment
* Hosted on Render

### Database

* PostgreSQL (Supabase)

### Media Storage

* Cloudinary (for image upload & delivery)

---

## 🔥 Features

* 📝 Create and manage blog posts
* 🖼️ Image upload with Cloudinary
* 🌐 REST API integration
* ⚡ Deployed full-stack (frontend + backend)
* 🔄 Real-world handling of CORS, environment configs, and deployment issues

---

## 🌍 Deployment Architecture

* Frontend → Netlify
* Backend API → Render
* Database → Supabase
* Media Storage → Cloudinary

---

## ⚠️ Note on Performance

This project is hosted on a free tier of Render:

* The backend may take a few seconds to respond if inactive
* This is due to **cold start (server sleep)** behavior
